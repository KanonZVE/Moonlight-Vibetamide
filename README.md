# 🌔 Moonlight Vibetamide

**Moonlight Vibetamide** es una evolución premium de Moonlight-Qt, rediseñada específicamente para ofrecer la mejor experiencia de streaming de juegos en dispositivos **Handheld** (ROG Ally, Steam Deck, Legion Go, MSI Claw) y sistemas basados en Linux como **Bazzite**.

Este fork introduce una capa de inteligencia adaptativa, optimizaciones de latencia críticas y una interfaz visual de primer nivel basada en *Glassmorphism*.

---

## ✨ Características "Vitaminadas" (Vibetamide)

### 🚀 Optimización para Handhelds
- **Smart Autoconfiguration**: El motor dinámico detecta tu hardware y red para ajustar automáticamente la resolución y el bitrate óptimo.
- **Platform-Aware**: Detección nativa de **Gamescope**, **Bazzite OS** y dispositivos como **ROG Ally**. Aplica automáticamente el modo *Borderless Windowed* y optimiza la entrada táctil (Trackpad Mode).
- **Ultra Low Latency (Vibetamide Pacer)**: Reduce el buffer de frames encolados y el buffer de audio Opus al mínimo absoluto para una respuesta instantánea.

### 🐔 Vibepollo Cloud Integration
- **Community Profiles**: Descarga automáticamente perfiles de configuración optimizados por la comunidad para cada juego y dispositivo específico desde la API global de Vibepollo.
- **Smart Handshake**: Sincronización transparente de capacidades entre el cliente y el host.

### 🌈 Visual Excellence (UX-UI Senior Redesign)
- **Glassmorphism Interface**: Un diseño moderno y translúcido con bordes redondeados y efectos de cristal.
- **Dynamic Backgrounds**: El fondo de la aplicación se transforma dinámicamente reflejando el arte del juego seleccionado con efectos de desenfoque y viñeta.
- **Micro-animations**: Transiciones de escala y efectos de iluminación (glow) al navegar por tu biblioteca de juegos.

### 💡 Mejoras Técnicas
- **Vulkan HDR10**: Parches específicos para corregir el soporte de 10-bit HDR en Linux/Vulkan, asegurando colores vibrantes y negros profundos.
- **Adaptive Bitrate**: Algoritmos mejorados para mantener la estabilidad del stream incluso en redes WiFi congestionadas.

## 🎮 Dispositivos Compatibles
Aunque funciona en cualquier PC, **Moonlight Vibetamide** brilla especialmente en:
- **ROG Ally / Ally X**
- **Steam Deck / OLED**
- **Lenovo Legion Go**
- **MSI Claw**
- Cualquier dispositivo ejecutando **Bazzite**, **SteamOS** o **ChimeraOS**.

2. Habilita **Ultra Low Latency Mode** para juegos competitivos.
